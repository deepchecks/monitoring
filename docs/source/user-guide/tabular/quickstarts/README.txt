Tabular Quickstarts
=========================

Here you can find quickstarts for easily onboarding your first model and data to deepchecks.
The :ref:`Get Deepchecks Monitoring Up and Running <quick_tabular>` is your go-to guide for the quickest
way to understand the basic concepts and send your own data to the system.

If you want a more detailed guide covering the data preparation, creating a data scheme, 
uploading reference data, etc., check out the :ref:`Preparing Your Tabular Data for Deepchecks Monitoring <prepare_your_tabular_data>`
guide.
